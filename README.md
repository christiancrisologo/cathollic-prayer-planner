
# Catholic Prayer Planner

A beautiful, interactive spiritual companion designed to help young Catholics develop a consistent prayer life. Built with React and powered by Google Gemini AI.

![Catholic Prayer Planner Banner](https://github.com/user-attachments/assets/0aa67016-6eaf-458a-adb2-6e31a0763ed6)

## ✨ Key Features

- **🙏 Daily Reflections**: Start each day with scripture and a short meditation. Cycle through standard orations or add your own favorite verses.
- **✨ AI Gospel Insights**: Get deep, spirit-led insights on today's Word, along with custom-generated prayers powered by **Gemini 2.0 Flash**.
- **⛪ Parish Finder**: Find local churches using geolocation. View mass schedules, get directions, and save your favorites.
- **📖 Prayer Library**: A curated collection of traditional prayers (Orationes) with a dedicated reading modal and **Text-to-Speech** support.
- **📅 Prayer Planner**: Schedule your spiritual commitments and receive notifications to stay on track.
- **🎨 Modern Aesthetics**: Choose between **Classic** and **Dark** themes, featuring a premium design with smooth animations and elegant typography.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- A Gemini API Key from [Google AI Studio](https://aistudio.google.com/)

### Local Installation

1.  **Clone the repository:**
    ```bash
    git clone [repository-url]
    cd prayer-planner
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Configure Environment:**
    Create a `.env` file in the root directory and add your Gemini API Key:
    ```env
    VITE_GEMINI_API_KEY=your_api_key_here
    ```

4.  **Run the Development Server:**
    ```bash
    npm run dev
    ```

## 🛠️ Tech Stack

- **Framework**: React 19 (Vite)
- **Styling**: Tailwind CSS
- **AI**: Google Generative AI (Gemini 2.0 Flash)
- **Typography**: Cinzel, Inter, Playfair Display
- **Icons**: Custom SVG + Lucide-style iconography
